A palindrome-Any is a string that has at least one character match when you flip it. In other words, we need the first letter equal the last letter, or the second letter must equal the second to last letter, or ... etc.
For example, "peek" is a palindrome-Any, because when you flip it, it becomes "keep". Both the second and third character match that of the original string.
