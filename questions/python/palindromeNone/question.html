A palindrome-None is a string that has no character match when you flip it. In other words, the first letter MUST NOT equal the last letter, and the second letter MUST NOT equal the second to last letter ... etc.
For example, "peak" is a palindrome-None, because when you flip it, it becomes "kaep". None of the characters match. But "peek" is not.
